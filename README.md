# ✨ PerfectYou™

**Become the Best Version of Yourself**

PerfectYou™ ist eine interaktive Installation in Form eines digitalen Spiegels.  
Das Projekt inszeniert ein fiktives KI-Startup, das verspricht, Schönheit, Glück und persönliches Potenzial anhand des Gesichts zu analysieren und zu optimieren.

Die Installation hinterfragt den gesellschaftlichen Drang zur Selbstoptimierung und die Idee, dass KI menschliche Eigenschaften objektiv messen oder verbessern kann.


## Konzept

Besucher*innen stellen sich vor eine Kamera und werden von einem vermeintlich intelligenten System analysiert.  
Das Interface zeigt FaceMesh, Scan-Animationen und verschiedene Scores wie **Happiness**, **Symmetry** oder **Perfect Face Score**.

Während die KI das Gesicht scheinbar optimiert, entfernt sich die Darstellung immer weiter vom ursprünglichen Aussehen. Am Ende entsteht eine künstliche, überperfekte und unmenschliche Version der Person.

> Je perfekter die KI dich macht, desto weniger bleibst du selbst übrig.


## Storyline

PerfectYou™ tritt als fiktives Unternehmen auf mit dem Versprechen:

**“Your face reveals your true potential.”**

Das System wirbt mit:

- Objective Facial Analysis
- Happiness Measurement
- Beauty Optimization
- Potential Prediction
- AI-powered Self Improvement

Die Installation ist bewusst wie ein professionelles Produktdemo gestaltet, um die Illusion eines echten KI-Tools zu erzeugen.


## Ablauf

### 1. Analyse

Ein Gesicht wird erkannt, FaceMesh und Scan-Animationen erscheinen.  
Texte wie `ANALYZING FACE...` oder `DETECTING EMOTIONAL PROFILE...` simulieren eine objektive Auswertung.

### 2. Bewertung

Das System zeigt verschiedene Kennzahlen:

- 🧴 glattere Haut
- 👁️ größere Augen
- 🪞 symmetrischere Gesichtszüge
- 🙂 angehobene Mundwinkel
- 😁 permanentes Lächeln

Die Scores werden als Balken und Prozentwerte dargestellt.

### 3. Optimierung

Das Gesicht wird schrittweise verändert:

- glattere Haut
- größere Augen
- symmetrischere Gesichtszüge
- angehobene Mundwinkel
- permanentes Lächeln

Am Ende stehen alle Werte auf 100 %.  
Das System meldet:

**`OPTIMIZATION COMPLETE`**  
**`YOU ARE NOW THE BEST VERSION OF YOURSELF`**


## Technische Umsetzung

## Setup
#### 1. Dependencies installieren

```bash
npm install
```
#### 2. Development Server starten
- Reicht zum starten des Projekts aus, vite aktualisiert automatisch wenn etwas geändert wird, obwohl der Server läuft
```bash
npm run dev
```
#### 3. Projekt builden
```bash
npm run build
```
## ⚠️ Hinweis
PerfectYou™ ist kein echtes Analyse- oder Bewertungssystem.
Alle Scores und Optimierungen sind Teil einer künstlerischen Inszenierung.
