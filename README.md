# PerfectYou

Interactive installation exploring AI-driven optimization, beauty standards and emotional evaluation.

## Setup

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build project:

```bash
npm run build
```

## Tech Stack

* HTML
* CSS
* JavaScript + Modules (p5.js, ml5, face-api.js)
* Vite
